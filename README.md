# Coffee Shop

## Descrição do projeto

O **Coffee Shop** é uma landing page desenvolvida como parte do bootcamp de Desenvolvimento Web. O projeto simula o site de uma cafeteria, apresentando informações sobre receitas de café, vídeos explicativos e um formulário para reserva de mesa.

A página foi construída utilizando **HTML e CSS**, com foco na estrutura semântica, organização do código e boas práticas de desenvolvimento front-end.

Entre as funcionalidades da página estão:

- Navegação com menu no topo da página
- Seção com vídeos incorporados sobre preparo de café
- Seção de reserva com formulário
- Rodapé com informações da cafeteria

## Tecnologias utilizadas

- **HTML5** – estrutura semântica da página
- **CSS3** – estilização da interface
- **Flexbox** – organização e alinhamento dos elementos no layout
- **Metodologia BEM** – organização e padronização das classes CSS
- **YouTube Embed (iframe)** – incorporação de vídeos na página

## Planos de melhoria

Durante o desenvolvimento do projeto, algumas ideias de melhorias foram identificadas para futuras versões da página:

- Tornar a página **responsiva para dispositivos móveis**, garantindo uma boa experiência em celulares e tablets.
- Implementar **JavaScript para validação do formulário de reserva**, tornando a interação mais dinâmica.
- Criar um **modo Dark (tema escuro)** para melhorar a acessibilidade e oferecer uma opção visual alternativa para os usuários.

Essas melhorias serão estudadas e implementadas em versões futuras do projeto como forma de aprofundar conhecimentos em desenvolvimento front-end.
